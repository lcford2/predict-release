NUMA Remote Assigned Mem: These are pages(MB) that overflow to another node when local memory is full.
NUMA All Mem Assignments: MBytes assigned with2MB pages (red); MBytes with 4K pages (blue).
              Usually best if all assigned memory is with 2MB pages
              (a ~5% variation may exist, due to non instantaneous kernel updates).
NUMA Free: Total Free memory on all nodes (color areas show node portions).
NUMA Used: Total Used memory on all nodes (color areas show node portions).
